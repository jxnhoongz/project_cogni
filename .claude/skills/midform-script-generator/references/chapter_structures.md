# Chapter Structure Frameworks

## Framework 1: Problem-Solution

```
1. THE PROBLEM (200 words)
   - Describe common struggle
   - Make viewer feel seen

2. THE SOLUTION (400 words)
   - Introduce book's key concept
   - Explain mechanism

3. THE EVIDENCE (500 words)
   - Story from book
   - Research or data
   - Real-world example

4. THE APPLICATION (300 words)
   - Specific steps to try
   - Common pitfalls to avoid
```

## Framework 2: Story-First

```
1. HOOK STORY (300 words)
   - Compelling narrative
   - Build tension/curiosity

2. THE PRINCIPLE (400 words)
   - Extract the lesson
   - Connect to book's thesis

3. COUNTER-EXAMPLE (300 words)
   - What happens when ignored
   - Contrast to reinforce

4. YOUR TURN (200 words)
   - Action step
   - Implementation tip
```

## Framework 3: Myth Buster

```
1. THE MYTH (200 words)
   - Common belief
   - Why people believe it

2. THE REALITY (500 words)
   - What research shows
   - Book's perspective

3. WHY IT MATTERS (300 words)
   - Stakes of getting it wrong
   - Benefits of truth

4. THE SHIFT (300 words)
   - How to change behavior
   - Practical next step
```

## Framework 4: Deep Dive

```
1. CONCEPT INTRODUCTION (300 words)
   - Define the idea
   - Why it's counterintuitive

2. MECHANISM (400 words)
   - How it works
   - Scientific basis

3. CASE STUDIES (500 words)
   - 2-3 brief examples
   - Diverse applications

4. IMPLEMENTATION (300 words)
   - Step-by-step guide
   - Tools or resources
```

## Transitions Between Chapters

- "But here's where it gets interesting..."
- "This leads to an important question..."
- "Now that you understand X, let's look at Y..."
- "The next piece of the puzzle..."
